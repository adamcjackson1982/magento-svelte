# Magento Svelte Theme
Magento 2 boiler plate theme with Svelte & Tailwind CSS. Just add yarn!

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn watch
```

### Add the following to config.php
```
'system' => [
            'design' => [
                'theme' => [
                    'theme_id' => 'frontend/Svelte/svelte-theme'
                ]
            ]
        ]
    ]
```
